Ajuda

Este documento especifica como usar o programa Restarante Universitário, referente ao projeto final da disciplina de Linguagem de Programação II ( LP33A ) , contida no terceiro semestre da matriz curricular do curso de Tecnologia de Desenvolvimento de Sistemas para Internet , ministrada pela professor Lucio Geronimo Valentin. O objetivo do documento é apresentar as tela e seus botões e suas funcionalidades.

Botão Debitar

Apertando o botão debitar da tela Principal vai abrir uma tela onde o funcionário vai fazer o débito da compra do cli ente colocando o valor e seu RA e o cliente digita a senha.

Apertando no debitar o programa vai automaticamente confirir se o cliente tem crédito, se ele não tiver crédito vai aparecer essa tela.

para conseguir debitar, o cliente terá que coloca crédito no seu RA.

Mas se tiver crédito vai aparecer essa tela.
