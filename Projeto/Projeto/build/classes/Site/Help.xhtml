Ajuda

Este documento especifica como usar o programa Restarante Universitário, referente ao projeto final da disciplina de Linguagem de Programação II (LP33A), contida no terceiro semestre da matriz curricular do curso de Tecnologia de Desenvolvimento de Sistemas para Internet, ministrada pela professor Lucio Geronimo Valentin. O objetivo do documento é apresentar as tela e seus botões e suas funcionalidades.

Botão Debitar

Apertando o botão debitar da tela Principal vai abrir uma tela onde o funcionário vai fazer o débito da compra do cliente colocando o valor e seu RA e o cliente digita a senha.

Apertando no debitar o programa vai automaticamente confirir se o cliente tem crédito, se ele não tiver crédito vai aparecer essa tela.

para conseguir debitar, o cliente terá que coloca crédito no seu RA.

Mas se tiver crédito vai aparecer essa tela.

E o valor será debitado do RA do cliente.

Botão Cliente

Apertando o botão Cliente da tela Principal vai abrir uma tela onde o funcionário vai ter quatro abas de opção sendo elas Cadastrar, Desativar, Alterar e Consultar.

Na aba Cadastrar o funcionário cadastra o clinete preenchendo todos os campos sendo RA, Nome, E-mail, Senha, Função (tem três opção Aluno, Professor, Funcionário), e o botão cadastrar para cadastrar o novo cliente, todos os campos tem que estar preencido para cadastrar.

Na aba Desativar no botão desativar o funcionário digita o RA do cliente que vai ser desativado, seus crédito será congelado sem poder creditar ou mesmo debitar, o cliente não será excluido.Já no botão ativar o funcionário digita o RA do cliente que está desativada e ativa a conta do cliente novamente.

Na aba alterar o funcionário digita o RA do cliente e aperta no botão alterar.

onde aparecer essa tela onde poderá altera os dados do cliente.

Na aba Consultar o funcionário digita o RA e clica no botão consultar onde aparecerá o Nome do cliente, E-mail e saldo.

a consulta vem assim.

Botão Creditar

Apertando o botão debitar da tela Principal vai abrir uma tela onde o funcionário vai inserir os crédito do valor que usuario solicitar .

Campo Consulta por Nome

Na Tela Principal tem um campo consulta na parte superior onde pode ser feito consulta de cliente escrevendo o nome completo vem dados do cliente como saldo conta ativa ou desativada.
